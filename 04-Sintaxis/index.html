Síntaxis

La sintaxis de JavaScript es el conjunto de reglas, cómo se construyen los programas de JavaScript:

  • valores: La sintaxis de JavaScript define dos tipos de valores:
    • Literales: Los valores fijos. Números, Cadenas,
    • Variables: Los valores variables
  • Operadores: + - * / =
  • Expresiones: Una expresión es una combinación de valores, variables y operadores, que se calcula en un valor. Las expresiones también pueden contener valores variables. Los valores pueden ser de varios tipos, como números y cadenas.
  • Palabras reservadas (keywords): Las palabras clave de JavaScript se utilizan para identificar acciones a realizar.
  • Comentraios: Código después de barras dobles //o entre /*y */se trata como un comentario. Los comentarios se ignoran y no se ejecutarán.
  • Sentencias: Finalizan con ;, Se puede escribir varias sentencias en al misma linea separandolas con ;, multiples espacios en blanco los omite el navegador.
