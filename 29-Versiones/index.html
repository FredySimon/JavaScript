Versiones

JavaScript fue inventado por Brendan Eich en 1995, y se convirtió en un estándar ECMA en 1997. ECMAScript es el nombre oficial del idioma. Desde 2015, ECMAScript se nombra por año(ECMAScript 2015).

Ediciones ECMAScript
Ver Nombre oficial Descripción
1 ECMAScript 1 (1997) Primera edición.
2 ECMAScript 2 (1998) Solo cambios editoriales.
3 ECMAScript 3 (1999) Se agregaron expresiones regulares. Añadido try / catch.
4 ECMAScript 4 Nunca lanzado.
5 ECMAScript 5 (2009) Se agregó "modo estricto".Se agregó soporte JSON.Se agregó String.trim ().Se agregó Array.isArray ().Se agregaron métodos de iteración de matriz.
5.1 ECMAScript 5.1 (2011) Cambios editoriales.
6 ECMAScript 2015 Añadido let y const. Se agregaron valores de parámetros predeterminados. Se agregó Array.find (). Se agregó Array.findIndex ().
7 ECMAScript 2016 Operador exponencial agregado (**). Se agregó Array.prototype.includes.
8 ECMAScript 2017 Se agregó relleno de cadena. Se agregaron nuevas propiedades de objeto. Fuciones asíncronas añadidas. Memoria compartida agregada.
9 ECMAScript 2018 Se agregaron propiedades de descanso / propagación. Se agregó iteración asincrónica. Promesa agregada.finalmente (). Adiciones a RegExp.

ECMAScript 3 es totalmente compatible con todos los navegadores.

ECMAScript 5 es totalmente compatible con todos los navegadores modernos .

JavaScript / ECMAScript

JavaScript:Fue desarrollado para Netscape. Netscape 2 fue el primer navegador en ejecutar JavaScript. Después de Netscape, la fundación Mozilla continuó desarrollando JavaScript para el navegador Firefox. La última versión de JavaScript fue 1.8.5. (Idéntico a ECMAScript 5).

ECMAScript:Fue desarrollado por ECMA International después de que la organización adoptó JavaScript. La primera edición de ECMAScript se lanzó en 1997.
